## to run:
### open the FormProject folder in vs code
### there will be two folders; one each for frontend and backend
### Frontend is reg-form folder
### Backend is RegFormApi folder
### In vs terminal change directory to reg-form or RegformApi; whichever you want to run
### For reg-form: type `ng serve` to start the local server (should be localhost 4200)
### For RegFormApi type `dotnet run` to start local server (localhost 5232)
## Both of them should be running at same time 
### to check if data is indeed going to backend; submit some details in the form and open "localhost:5232/api/users" and check
